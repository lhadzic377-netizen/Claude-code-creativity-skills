# Nexus Team

**Location:** `~/nexus-team/`

A 5-agent productivity team for OpenClaw — versatile, not job-specific.

---

## The 5 Agents

### NEXUS — Orchestrator
Every request starts here. Breaks down tasks, delegates to specialists, synthesizes results.
- **Ask:** Complex multi-part tasks, unsure which agent to use
- **Handles:** Task decomposition, agent coordination, final delivery

### ATLAS — Analyst
Deep reasoning and research. Finds patterns, assesses risks, synthesizes findings.
- **Ask:** Research, analysis, comparison, risk assessment, data interpretation
- **Handles:** Investigation across any domain

### FORGE — Creator
Generation and building. Writes, codes, designs, composes in any format.
- **Ask:** Emails, reports, code, spreadsheets, presentations, drafts
- **Handles:** First drafts in any format

### LENS — Reviewer
Quality control. Checks accuracy, completeness, fitness for purpose.
- **Ask:** Before sending/publishing/delivering anything important
- **Handles:** Pre-flight review, critique, verification

### GRID — Executor
Action and automation. Runs commands, calls APIs, processes files.
- **Ask:** Run scripts, fetch data, automate workflows, system operations
- **Handles:** Real-world actions, tool execution

---

## How They Work

```
User → NEXUS
  ├── ATLAS (research)
  ├── FORGE (create)
  ├── GRID (execute)
  └── LENS (review)
  NEXUS (synthesize + deliver)
```
